# 🚀 快速上手教學 (白話版)

不用懂程式碼，只要 3 個步驟就能把 Word 變成網頁 HTML！

---

## 第 1 步：啟動程式 🟢

1. 打開你的資料夾：`photonic-filament` (你現在的這個資料夾)。
2. 找到一個叫做 **`start.bat`** 的檔案。
3. **點兩下** 執行它。

👉 **發生了什麼事？**

- **第一次執行會跑比較久**，因為要幫你安裝需要的工具 (Python 套件)，請耐心等待一下 (大約 1-2 分鐘)。
- 之後會跳出兩個黑色的視窗，這是正常的！
- 一個是 **後端** (負責運算的腦袋)
- 一個是 **前端** (讓你操作的介面)
- **請不要關掉這兩個黑色視窗**，縮小放到旁邊就好。

---

## 第 2 步：打開網頁介面 🌐

1. 正常來說，網頁會自動打開。
2. 如果沒有自動打開，請打開你的瀏覽器 (Chrome, Edge 都可以)。
3. 在網址列輸入這個網址：
   👉 **<http://localhost:7860>**

---

## 第 3 步：開始轉檔 📝

### A. Word 轉 HTML

看到的畫面應該是黑底的，左邊有一些選項：

1. **上傳檔案**：點擊框框，選擇你寫好的 Word (`.docx`) 檔。
   - *小撇步：Word 裡面的 YouTube/IG 連結只要「單獨一行」，轉出來就會自動變成影片卡片喔！*
2. **選擇分類**：看這篇文章是哪個主題 (例如「數據分析解方」)，選一下。
   - *這會決定文章最下面要放哪一種廣告/聯絡資訊 (Footer)。*
3. **按下「開始轉換」**：點那個橘色的按鈕。
4. **下載**：右邊會出現結果，點 **「下載 HTML 檔案」** 就搞定了！

### B. 圖片處理 (如果需要)

如果你有圖片想要縮小一點放網站：

1. 點上面分頁的 **「圖片處理」**。
2. 上傳你的大圖。
3. 直接按 **「開始處理」** (預設設定通常就很好用了)。
4. 下載處理好的圖片。

---

## ❓ 常見問題

**Q: 那兩個黑色視窗很礙眼，可以關掉嗎？**
A: **不行喔！** 關掉的話程式就停止運作了。用完之後再關掉就好。

**Q: 轉出來的 HTML 要怎麼用？**
A: 把下載下來的 `.html` 檔案交給工程師，或是用記事本打開，複製裡面的程式碼貼到後台。

**Q: 我的 IG 連結怎麼沒有變卡片？**
A: 請確認在 Word 裡面，那個連結是 **「單獨自己一行」**，前後不要有字。
